🛠️ 安装与使用
环境要求
Python 3.8+ / Java 11+

MySQL 8.0+ / Redis 6.0+

OS: Windows 10+, Linux (CentOS 8+/Ubuntu 20.04+), MacOS 12+（适配主流云平台部署环境，如字节、华为云）

快速启动
bash
# 1. 克隆项目
git clone https://github.com/Willy-WeiPengchen/copyright-ownership-dictionary.git

# 2. 进入目录
cd copyright-ownership-dictionary

# 3. 安装依赖 (Python示例)
pip install -r requirements.txt

# 4. 初始化数据库
python manage.py init-db

# 5. 启动服务
python manage.py runserver
